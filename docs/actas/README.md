# Actas de Reunión — PetLink

Esta carpeta contiene las minutas y actas generadas durante el proyecto.

## Formato de Actas

Nombre de archivo: `YYYY-MM-DD-acta-[tema].md`

## Campos Obligatorios por Acta

- Fecha y hora
- Participantes
- Puntos tratados
- Acuerdos tomados
- Responsables y fechas compromiso
